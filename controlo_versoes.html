Comandos GIT

iniciar o repositorio local: git init

ver o status: git status

para adicionar um ficheiro para o staging area (area de preparacao): git add nomeficheiro
git add ficheiro1 ficheiro2
para adicionar todos os ficheiros e pastas no staging area git add.

para gravar um snapshot do programa: git commit -m "pequena descricao"

Configuracao login GIT

git config --global user.name "username"

git config --global user.mail "email"
