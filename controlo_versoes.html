Comandos GIT

iniciar o repositorio local: git init

ver o status: git status

para adicionar um ficheiro para o staging area (area de preparacao): git add nomeficheiro
git add ficheiro1 ficheiro2
para adicionar todos os ficheiros e pastas no staging area git add.

Remover ficheiro da staged area sem remover do sistema: git reset nomeficheiro
ou
git restore --staged nomeficheiro

Gravar snapshot do programa: git commit -m "pequena descricao"

listar detalhadamente as versoes: git log

listar resumidamente as versoes: git log --oneline --graph

Configurar identidade

git config --global user.name "username"

git config --global user.mail "email"

Ramificacoes do codigo

listar os ramos: git branch

criar nova ramificacao/branch:git branch nomeDoRamo

Para mudar de ramo: git switch nomeDoRamo
